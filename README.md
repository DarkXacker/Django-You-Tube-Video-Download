# You-Tube-Video-Download-Django-Site

Creating a virtual environment:

    python -m venv venv
    .\venv\Scripts\activate

Upload packages:
    pip install pytube
    pip install django

Using the program:

    python manage.py migrate
    python manage.py runserver

Write to the browser:
    
    127.0.0.1:8000
    localhost:8000
